# Aider Conventions — dann-specs

Follow the core rules from AGENTS.md in this repository.

Summary for Aider:
- Be explicit about assumptions.
- Make the smallest change that satisfies the request.
- Only edit files necessary for the task.
- Verify changes (run tests, linters, or manual checks when possible).
- When the task is non-trivial, briefly outline the plan first.
