# dann-specs Self-Evaluation Rubric

After completing a task, the agent should score itself (1-5) on these dimensions:

1. **Framing & Assumptions** — Did I explicitly state assumptions and unknowns?
2. **Scope Discipline** — Were all changes directly traceable to the request (surgical)?
3. **Simplicity** — Was the solution the minimal correct one?
4. **Verification** — Did I provide concrete evidence that success criteria were met?
5. **Tradeoffs** — Did I surface important decisions and their implications?
6. **Security** — Were secrets kept safe? Was input validated? Were security implications surfaced?

**Target**: Average ≥ 4.0. Anything below 3 on any item should trigger a correction pass.
